# 규칙 괴담 장르 템플릿 (Rules Mystery / Honkaku)

> **핵심 셀링포인트**: 논리 추리 + 공정한 단서 + 지략 두뇌 싸움. 독자와 탐정이 같은 출발선에 서며, 진실은 반드시 추론 가능해야 한다.

---

- 추천 제약 팩: Pack M14 / Pack M06
- 공통 추가 팩: Pack U02

## 1. 핵심 세부장르 분류

### 본격 추리 계열
- **특징**: 순수한 논리 추리, 밀실 / 트릭 / 알리바이
- **핵심 사이다**: 수수께끼 해독의 쾌감, 진실 공개, 범인 처벌
- **대표 구조**: 사건 발생 → 수사 및 증거 수집 → 추리와 폭로

### 규칙 괴담 계열
- **특징**: 도시 전설, 기이한 규칙, 생존 해결
- **핵심 사이다**: 규칙 발견, 규칙 활용, 살아서 탈출
- **대표 구조**: 괴담 조우 → 규칙 발견 → 규칙을 이용해 탈출

### 서스펜스 스릴러 계열
- **특징**: 분위기 연출, 심리적 공포, 끊임없는 반전
- **핵심 사이다**: 긴박한 스릴, 충격적인 진실, 긴 여운
- **대표 구조**: 서스펜스 시작 → 겹겹이 벗겨짐 → 충격적 반전

### 사회파 추리 계열
- **특징**: 범죄 동기 탐구, 사회 문제 비판
- **핵심 사이다**: 인간 본성의 깊이, 사회 성찰, 감정적 공감
- **대표 구조**: 사건의 이면 → 인간 본성 탐구 → 사회 비판

## 2. 본격 추리 십계（웹소설 적용 버전）

```
제1계: 범인은 이야기 초반에 반드시 등장해야 한다
제2계: 초자연적 힘은 진실 해명에 사용 금지（미스리딩에는 사용 가능）
제3계: 밀실에는 비밀 통로가 최대 하나만 허용된다
제4계: 미지의 독약이나 장황한 과학적 설명이 필요한 장치는 사용 금지
제5계: 전혀 등장하지 않은 신비로운 인물을 범인으로 삼을 수 없다
제6계: 탐정은 우연이나 육감에 의존해 사건을 해결할 수 없다
제7계: 탐정 자신이 범인일 수 없다（더블 탐정 설정 제외）
제8계: 탐정은 모든 단서를 독자에게 공개해야 한다
제9계: 조수（왓슨 역할）의 사고 과정은 독자에게 투명해야 한다
제10계: 쌍둥이 / 분신은 초반에 반드시 명시되어야 한다
```

## 3. 4대 핵심 기둥

### 기둥 1: 공정성 (Fair Play)
```
✅ 올바른 예:
10화: 탐정이 꽃병 파편을 발견（독자도 동시에 확인）
20화: 탐정이 "꽃병은 내부에서 깨졌다"고 추리
→ 독자가 소급하여 검증 가능

❌ 잘못된 예:
10화: 탐정이 단서를 발견（내용 미서술）
20화: 탐정이 "나는 진작에 핵심 증거를 발견했다!"고 말함
→ 독자가 검증할 수 없으므로 불공정
```

### 기둥 2: 논리성 (Logic)
```
논리 체인 예:

알려진 단서:
1. 피해자는 잠긴 방 안에 있음
2. 창문은 내부에서 잠겨 있음
3. 피해자가 열쇠를 쥐고 있음

추리 과정:
1. 피해자가 열쇠를 쥐고 있음 → 그가 직접 잠갔을 가능성
2. 창문이 내부에서 잠겨 있음 → 범인이 창문으로 도망치지 않음
3. 방이 잠겨 있음 → 범인은 피해자가 문을 잠그기 전에 이미 떠났을 가능성
4. 추측: 피해자가 문을 잠근 뒤, 미리 방 안에 숨어 있던 범인에게 살해당함
5. 검증: 방 안에서 숨을 수 있는 장소 탐색
```

### 기둥 3: 해결 가능성 (Solvability)
```
단서 배치 원칙:
- 핵심 단서는 최소 3회 등장（다른 형태로）
- 레드 헤링（미스리딩）은 실제 단서 수를 초과할 수 없음
- 폭로 전 반드시 "마지막 힌트"를 제시해야 함（challenge to reader）
```

### 기둥 4: 의외성 (Surprise)
```
반전 단계:
Level 1: 범인 신분 반전
Level 2: 동기 반전
Level 3: 사건 본질 반전
Level 4: 서술자 / 시점 반전
```

## 4. 규칙 괴담 프레임워크

### 규칙 설계 원칙
```
1. 규칙에는 내재적 논리가 있어야 한다
2. 규칙의 예외에는 복선이 필요하다
3. 규칙을 이용한 탈출은 합리적이어야 한다
4. 규칙의 기원은 설명이 필요하다（신비로울 수 있지만 암시는 필요）
```

### 규칙 괴담 구성
```
조우 단계 (20%):
- 주인공이 이상한 환경에 진입
- 첫 번째 피해자 / 이상 징후 발견
- "규칙 안내서" 또는 구두 힌트 획득

탐색 단계 (40%):
- 규칙의 경계를 시험
- 누군가 규칙을 어겨 처벌받음
- 규칙 체계를 점차 이해

활용 단계 (30%):
- 규칙의 허점 발견
- 탈출 계획 수립
- 실행 과정에서의 의외 변수

결말 단계 (10%):
- 탈출 성공 또는 실패
- 규칙의 최종 해명
- 여운（개방적 암시）
```

### 규칙 예시
```markdown
【엘리베이터 규칙】
1. 자신이 가려는 층만 누를 수 있다
2. 엘리베이터 안에 사람이 있을 때는 말하지 않는다
3. 거울 속에 여분의 인영(人影)이 보이면 즉시 눈을 감는다
4. 엘리베이터가 4층에 멈추면 내리지 않는다

【규칙 활용】
- "말하면 안 된다"는 규칙을 이용해 동료에게 위험을 암시
- "거울"을 이용해 변장한 자를 발견
- "4층 금지 구역"을 이용해 적을 유인
```

## 5. 플롯 구성（40만 자 기준）

### 1권: 사건 발생 (1~30화, 15%)
- **오프닝 훅**: 시체 / 이상 징후 / 실종
- **주요 인물 등장**: 탐정, 용의자, 피해자 관계인
- **환경 설정**: 밀폐 공간 / 시간 제한 / 특수 규칙
- **1차 조사**: 기초 단서 수집

### 2권: 수사 (31~100화, 35%)
- **용의자 정리**: 각자의 동기, 기회, 알리바이
- **단서 축적**: 물증, 인증, 정보
- **미스리딩 전개**: 레드 헤링 등장
- **소클라이맥스**: 두 번째 사건 / 새로운 발견

### 3권: 추리 (101~150화, 25%)
- **단서 통합**: 연관 분석
- **가설 검증**: 배제법 / 확인법
- **진실 접근**: 핵심 단서 부상
- **독자에게 도전**: 마지막 힌트 제시

### 4권: 폭로 (151~180화, 15%)
- **탐정 추리**: 완전한 논리 체인
- **범인 폭로**: 신분 확인
- **동기 공개**: 범죄 이유
- **트릭 해설**: 수법 상세 설명

### 5권: 마무리 (181~200화, 10%)
- **범인 처벌**: 법적 / 도덕적 심판
- **인물 결말**: 각자의 운명
- **여운**: 성찰 / 새 사건 암시

## 6. 단서 설계 기법

### 3층 단서법
```
1층: 표면 단서（모두가 볼 수 있음）
- 사망 시각, 장소, 방식
- 명확한 물증

2층: 숨겨진 단서（주의를 기울여야 발견 가능）
- 무심코 지나친 디테일
- 대화 속 모순
- 타임라인의 빈틈

3층: 심층 단서（연상 / 전문 지식 필요）
- 행동 심리 분석
- 전문 분야 지식
- 역사 / 배경 연관성
```

### 레드 헤링 설계
```
유형 1: 가짜 용의자
- 동기는 있지만 기회가 없음
- 현장에 있었지만 알리바이 존재

유형 2: 가짜 증거
- 위조된 물증
- 조작된 증인

유형 3: 가짜 추리
- 조수의 오류 추단
- 탐정의 의도적 미스리딩

유형 4: 가짜 신분
- 변장자
- 사칭자
```

## 7. 용의자 관리

### 용의자 설계 템플릿
```
이름: ___
관계: 피해자와의 관계
동기: 살인을 저질러야 했던 이유
기회: 범행 시간이 있었는지 여부
수단: 범행 능력이 있었는지 여부
알리바이: 있음 / 없음 / 의심스러움
비밀: 숨겨진 정보（반드시 사건과 관련 없어도 됨）
```

### 용의자 수 권장
```
단편 (5만 자 이하): 3~5명
중편 (5~20만 자): 5~8명
장편 (20만 자 이상): 8~12명
```

## 9. 클래식 트릭 분류

### 밀실 트릭
```
1. 기계식 잠금（타이머 / 원격 / 자동）
2. 사전 잠입 후 대기
3. 피해자가 직접 문을 잠근 뒤 살해
4. 방이 밀실이 아님（착각）
5. 사망 시각 위조
```

### 알리바이 트릭
```
1. 시각 위조（시계 / 녹화 영상）
2. 대역 변장
3. 쌍둥이 / 유사인물
4. 원격 살인
5. 공범 협력
```

### 신분 트릭
```
1. 사망자가 사망하지 않음（위장 사망）
2. 범인이 범인이 아님（누명 씌워짐）
3. 탐정이 곧 범인
4. 서술자의 기만
```
