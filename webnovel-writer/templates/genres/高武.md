# 고무 장르 템플릿 (고무/현대 무력, High Martial Arts)

> **핵심 셀링포인트**: 무도의 극치 + 열혈 전투 + 실력이 곧 진리. 주먹이 도리(道理)이며, 강자가 규칙을 만든다.

---

## 창의 제약 추천 (선택)
- 추천 제약 팩: Pack M21
- 공통 추가 팩: Pack U03

## 1. 핵심 계열 세부장르

### 전 세계 고무류
- **특징**: 영기 부활, 전 인류가 무술을 익히며 무도가 주류가 됨.
- **핵심 사이다**: 무도 돌파, 랭킹 상승, 국가대전 영광.
- **대표작**: 《전 세계 고무》《나에게는 검이 있다》《만족의 겁》

### 무도 종사류 (Grandmaster)
- **특징**: 전통 무술, 종사(宗師) 경지, 무학 전승.
- **핵심 사이다**: 무학 돌파, 문파 패권, 무림 맹주.
- **대표작**: 《용사연의》《국술종사》

### 격투 경기류
- **특징**: 무대 비무, 프로 격투, 열혈 대결.
- **핵심 사이다**: 링 위의 승리, 기술 성장, 챔피언의 영광.
- **대표작**: 《권진산하》《전직고수》

### 무도 수진류 (Martial Cultivation)
- **특징**: 무도와 수련을 결합, 무도로 도(道)에 입문.
- **핵심 사이다**: 무도 경지, 육신 성인(成聖), 무도로 허공을 부숨.
- **대표작**: 《차천》《완벽한 세계》《성허》

## 2. 세계관: 고무 사회학

### 무도의 위상
- **은밀기**: 무자(武者)들이 숨어 있고, 일반인은 존재를 모름.
- **공개기**: 무도가 공개되고 하나의 직업으로 자리 잡음.
- **주류기**: 무도가 사회 핵심이 되고, 무자의 지위가 높아짐.

### 사회 구조
| 계층 | 특징 | 권력 |
|------|------|------|
| 무성(武聖) | 한 나라의 전력 | 초월적 지위 |
| 종사(宗師) | 나라를 지키는 전력 | 국사급 |
| 대사(大師) | 한 도시의 전력 | 고위층 |
| 고수(高手) | 엘리트 전력 | 중간층 |
| 무자(武者) | 기초 전력 | 일반 |
| 일반인 | 무력 없음 | 하층 |

### 세력 구도
- **국가 무도부**: 공식 무자 조직.
- **무도 세가**: 오랜 전통을 지닌 무학 가문.
- **무관/도장**: 무학 전수 기관.
- **지하 세력**: 암시장, 킬러 조직.
- **이족/요수**: 인류의 적.

## 3. 무도 체계 설계

### 경지 구분 (표준판)
| 경지 | 특징 | 전투력 |
|------|------|--------|
| 명경(明勁) | 힘을 밖으로 방출 | 일 대 열 |
| 암경(暗勁) | 힘을 안으로 집중 | 일 대 백 |
| 화경(化勁) | 힘을 허(虛)로 화함 | 일 대 천 |
| 종사(宗師) | 영역 초기 형성 | 일 대 만 |
| 대종사(大宗師) | 영역 대성(大成) | 도시 하나의 전력 |
| 무성(武聖) | 무도의 극치 | 나라 하나의 전력 |

### 무학 분류
| 종류 | 특징 | 대표 |
|------|------|------|
| 외공(外功) | 육신 강화 | 금종조(金鐘罩)·철포삼(鐵布衫) |
| 내공(內功) | 내력 수련 | 구양신공(九陽神功) |
| 권법(拳法) | 근접 격투 | 형의권(形意拳)·팔극권(八極拳) |
| 퇴법(腿法) | 하체 공격 | 담퇴(譚腿)·착각(戳脚) |
| 병기(兵器) | 무기 기법 | 검법·도법 |
| 경공(輕功) | 신법과 속도 | 능파미보(凌波微步) |

### 무도 자원
- **무학 비급**: 공법 전승.
- **단약**: 체질 강화, 부상 치료.
- **천재지보**: 돌파구 마련.
- **무기**: 신병이기(神兵利器).

## 4. 클래식 사이다 클리셰

### 링 위 압도
- **장면**: 공식 비무, 모두가 지켜보는 앞에서.
- **사이다**: 단 일격에 제압, 전장을 경악시킴.
- **핵심**: 충분한 복선과 상대 캐릭터 구축 필수.

### 무학 돌파
- **장면**: 생사의 기로/폐관 수련.
- **사이다**: 경지 돌파, 실력 급상승.
- **핵심**: 돌파는 축적이 있어야 하며 갑자기 이루어질 수 없음.

### 주먹으로 설득 (이무복인, 以武服人)
- **장면**: 무시당하다가 실력으로 말함.
- **사이다**: 주먹이 도리, 모두를 굴복시킴.
- **핵심**: 무분별한 면상 후려치기 금지. 절제 있게.

### 국가대전 영광
- **장면**: 국가 대표로 출전.
- **사이다**: 나라를 위한 명예, 민족적 자부심.
- **핵심**: 가국(家國)에 대한 감정선 복선 필수.

## 5. 대강 구조 (200만 자 표준)

### 1권: 입문 (1~80장)
- **핵심**: 주인공의 무도 재능 각성.
- **무대**: 도시/무관.
- **클라이맥스**: 첫 번째 재능 발현, 동급 강적 격파.

### 2권: 부상 (81~180장)
- **핵심**: 무도 조직 가입, 대회 참가.
- **무대**: 지방/전국 규모.
- **클라이맥스**: 주요 대회 우승.

### 3권: 이름을 날리다 (181~350장)
- **핵심**: 유명 무자로 성장.
- **무대**: 전국 범위.
- **클라이맥스**: 종사 돌파, 무림을 경악시킴.

### 4권: 패권 다툼 (351~550장)
- **핵심**: 국제 경쟁 참가.
- **무대**: 국제 범위.
- **클라이맥스**: 국가대전 승리, 국위 선양.

### 5권: 정점 (551~700장)
- **핵심**: 무도의 극치 추구.
- **무대**: 전 세계/이계(異界).
- **클라이맥스**: 무도 정점 달성.

## 6. 전투 설계

### 전투 리듬
```
탐색 단계 (20%):
- 양측이 상대방을 탐색하며 기본 실력 노출
- 관중/해설이 양측 배경 소개

교전 단계 (40%):
- 본격 교전, 초식(招式) 맞대결
- 무학의 특색 드러냄

클라이맥스 단계 (30%):
- 필살기 대결
- 반전 연출

마무리 단계 (10%):
- 승패 확정
- 이후 파급 효과
```

### 전투 묘사 핵심
- **초식 이름**: 무협감 있게 작명.
- **전력 비교**: 구체적인 묘사 필수.
- **환경 상호작용**: 전투가 주변 환경에 미치는 영향.
- **심리 묘사**: 양측의 심리 변화.

## 7. 무도 세가 설계

### 세가 핵심 요소
| 요소 | 설명 |
|------|------|
| 전승 무학 | 가문 고유의 무학 |
| 가문 역사 | 여러 세대에 걸친 축적 |
| 자원 비축 | 단약·비급·천재지보 |
| 인재 양성 | 가문 자제 양성 체계 |
| 대외 관계 | 다른 세력과의 관계 |

### 세가 갈등
- **자원 쟁탈**: 천재지보 쟁탈.
- **랭킹 경쟁**: 세가 순위 다툼.
- **은원(恩怨)**: 역사적으로 남겨진 문제.
- **정략결혼**: 가문 혼인을 둘러싼 암투.

## 8. 흔한 지뢰 (금지 사항)

### 반드시 피할 것
- 전투력 붕괴, 수치 인플레이션
- 면상 후려치기가 너무 잦아 피로감 유발
- 무학 설정의 전후 모순
- 조연의 소도구화(공기 취급)

### 권장 방향
- 전투력 체계의 자기 일관성 유지
- 면상 후려치기는 절제하고 복선을 깔 것
- 무학 설정을 상세하고 일관되게
- 조연에게도 독자적인 인격 부여
